恭喜，LAMP一鍵安裝懶人包安裝成功！

查看本機環境： 探針 phpinfo phpSysInfo SysInfo VPSInfo phpMyAdmin

LAMP使用方法：

  • lamp (add,del,list)：創建（刪除，列出）虛擬主機。

  • lamp ftp(add|del|list)：創建（刪除，列出）ftp用戶。

  • lamp uninstall：一鍵刪除lamp（刪除之前注意備份好網站資料！）。

    • 程式資料夾：

  • MySQL安裝資料夾：/usr/local/mysql

  • MySQL資料庫資料夾：/usr/local/mysql/data（預設，安裝時可更改路徑）

  • MariaDB安裝資料夾：/usr/local/mariadb

  • MariaDB資料庫資料夾：/usr/local/mariadb/data（預設，安裝時可更改路徑）

  • PHP安裝資料夾：/usr/local/php

  • Apache安裝資料夾：/usr/local/apache

    • 指令一覽：

  • MySQL 或 MariaDB 指令: /etc/init.d/mysqld(start|stop|restart|status)

  • Apache 指令: /etc/init.d/httpd(start|stop|restart|status)

    • 網站根資料夾：

  • 預設網站根資料夾： /data/www/default
